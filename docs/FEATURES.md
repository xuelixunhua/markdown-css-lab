# Features

## 本地渲染器

- Markdown 编辑：左侧直接输入文章内容。
- CSS 编辑：可查看并修改当前主题 CSS；手动修改后会作为自定义 CSS 保存。
- 实时预览：右侧使用公众号近似宽度展示最终效果。
- 主题切换：公众号、PDF 手册、技术文档三套主题可切换。
- 同步滚动：编辑区滚动时预览区按比例跟随。
- 目录生成：使用 `[TOC]` 自动生成紧凑目录。
- 外链脚注：`http://` / `https://` 外部链接默认转成正文上标编号，并在文末生成带标题的“参考链接”。
- 富文本复制：复制带 inline style 的 HTML，供微信公众号后台粘贴。
- HTML 复制与导出：便于调试、归档和后续自动化。

## CSS 主题

- `themes/wechat-editorial.css`：公众号长文，强调移动端阅读、段落节奏和重点突出。
- `themes/founder-editorial.css`：PDF 手册，强调版心、留白、层级和纸面感。
- `themes/technical-doc.css`：技术文档 / 研究笔记，强调结构密度、表格、代码和引用。

## 微信兼容处理

- 复制时把 CSS 计算结果转成 inline style。
- 复制时移除 `width` / `max-width` / `height` 等容易导致微信后台变窄的尺寸。
- 复制时把伪元素装饰转成真实元素。
- 复制时把强调标签转成 `span`，减少微信编辑器对语义标签的二次改写。
- 复制时把列表项开头强调词后的冒号绑定到强调节点里，避免孤立冒号换行。
- 渲染前自动合并定义式列表项：`1. **标题**:\n正文` 会变成 `1. **标题**：正文`。
- 渲染前自动合并说明式列表项：`- **标题**\n——说明` 会变成 `- **标题**——说明`。
- 复制时清理列表项开头标签后的 `<br>`，避免公众号后台保留强制换行。
- 复制到公众号时会把真实 HTML 列表转换成手写符号列表，避免微信编辑器把加粗标题和后续正文拆成两行。
- 外部链接复制时不保留可点击外链，而是复制为脚注编号和文末 URL，适配微信公众号外链限制。
- `mp.weixin.qq.com` 这类微信内链会保留在正文里，不进入外链脚注。
